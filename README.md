# Lesson 18

Навчальний адаптивний лендінг, зібраний на `Pug`, `SCSS` і `Gulp`.

Проєкт демонструє базову верстку з кількома секціями, адаптивну сітку, мобільне меню та збірку статичних файлів у готовий HTML/CSS формат.

## Опис

Сторінка складається з таких блоків:

- header з навігацією та логотипом
- hero-секція з головним заголовком і кнопками
- skills-секція з картками навичок
- about-секція з текстом та ілюстрацією
- subscribe-секція з формою
- posts-секція з картками публікацій
- footer з соціальними посиланнями

## Технології

- HTML5
- Pug
- SCSS
- JavaScript
- Gulp 5
- PostCSS
- Autoprefixer
- cssnano
- BrowserSync

## Можливості

- адаптивна верстка під різні екрани
- збірка Pug-шаблонів у HTML
- компіляція SCSS у CSS
- мініфікація CSS
- автоматичне додавання префіксів
- локальний сервер із live reload
- мобільне бургер-меню

## Структура проєкту

- `src/templates/` - Pug-шаблони сторінки
- `src/scss/` - вихідні стилі SCSS
- `assets/css/` - зібрані CSS-файли
- `assets/js/` - JavaScript
- `assets/images/` - зображення та іконки
- `src/fonts/` - шрифти
- `gulpfile.js` - задачі збірки
- `index.html` - згенерована сторінка

## Встановлення

1. Встановіть залежності:

```bash
npm install
```

2. Запустіть збірку та локальний сервер:

```bash
npx gulp
```

## Команди Gulp

- `npx gulp` - базова збірка SCSS, мініфікація CSS, компіляція Pug і запуск watcher'а
- `npx gulp --dev` - режим розробки з sourcemaps
- `npx gulp --css` - компіляція Pug і спостереження за CSS

## Примітка

Це навчальний проєкт, тому структура і рішення в ньому орієнтовані на демонстрацію базових підходів до верстки та збірки фронтенду.
